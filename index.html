Data Set 1: NFL Elo Ratings¶

The NFL Elo ratings dataset was created by fivethirtyeight.com and assigns a power rating (around 1500 is average) for each team that help determine win probability for a given matchup. The prediction takes into account quality of teams, location of game played, strength of quarterback, and more. In the dataset, there are variables showing each team, the date the game was played, the Elo ratings for each team before and after the game, the home away designations, and whether the game was in the playoffs (and more).

Data Set 2: NFL Stadiums¶

The nfl stadiums dataset contains data on ~120 stadiums, 30 of which are nfl stadiums. It has useful weather data on these stadiums such as the typical temperature as well as if stadiums are indoor, outdoor, or retractable, as well as grass type. These are interesting metrics to look at and see how much these factors might affect game outcomes or points scored

Data Set 3: NFL Teams¶

This dataset does not contain data with relavent predicting power, but it contains a mapping from nfl team names to their abbreviations, which is useful in merging our datasets on the same team_name index

Data Set 3: Spreadspoke Scores¶

This dataset contains data results from NFL games dating back to 1966 as well as betting odds dating back to 1979. The dataset draws information not only from the results of each NFL game, but meaningful statistics, predictions, weather factors, and stadium data that help tell a complete story on each result. Additionally, the data helps paint a picture on successful betting strategies employed historically, which could be helpful in building a more successful predictive betting model for NFL games.

# create function that calculates whether the favor/underdog covers
def calculate_winner(row):
    if row['historical_spread'] < 0: # home team is favorite
        if row['home_score'] - row['away_score'] > abs(row['historical_spread']):
            return 1 # favorite covers
        else:
            return 0 # favorite doesn't cover
    elif row['historical_spread'] > 0: # away team is favorite
        if row['away_score'] - row['home_score'] > abs(row['historical_spread']):
            return 1 # underdog covers
        else:
            return 0 # underdog doesn't cover
    else:
        return 2 # tie

df_spread_comparison['outcome'] = df_spread_comparison.apply(calculate_winner, axis=1)
